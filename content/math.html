Kinematic Equations

Conventions

\bar{v} &= \text{Average velocity}\\ \bar{a} &= \text{Average acceleration}\\ \Delta{x} &= x_2 - x_1\\ \Delta{v} &= v_2 - v_1\\ v_2 &= v_1 + a\Delta{t} (v_2)^2 &= (v_1)^2 + 2a\Delta{x} \Delta{x} &= v_1 \Delta{t} + \frac{1}{2}a\Delta{t^2}\\ &= v_2\Delta{t} - \frac{1}{2}a\Delta{t^2}\\ &= \left(\frac{v_2 + v_1}{2}\right)\Delta{t}

Basics

Constant Velocity

\bar{v} &= \frac{\Delta{x}}{\Delta{t}}\\ \Delta{x} &= \bar{v}\Delta{t}

Uniform acceleration

\bar{a} &= \frac{\Delta{v}}{\Delta{t}}\\ v_2 &= v_1 + a\Delta{t}\\ \Delta{x} &= v_0\Delta{t}+\frac{1}{2}a\Delta{t^2}\\ (v_2)^2 &= (v_1)^2 - 2\bar{a}\Delta{x}

Miscellaneous

\bar{v} &= \frac{\Delta{x}}{\Delta{t}} = \frac{v_2 + v_1}{2}\\ \bar{a} &= \frac{\Delta{v}}{\Delta{t}}

Displacement

Displacement when object moves with constant velocity

\Delta{x} &= \bar{v}\Delta{t}

Displacement when object accelerates from rest

\Delta{x} &= \frac{1}{2}\Delta{v}\Delta{t}\\ &= \frac{1}{2}\bar{a}\Delta{t^2}

Displacement when object accelerates with initial velocity

\Delta{x} &= v_1\Delta{t} + \frac{1}{2}\Delta{v}\Delta{t} \\ &= v_1\Delta{t} + \frac{1}{2}\bar{a}\Delta{t^2}

Deriving Kinematic Formulas

Given

\bar{a} &= \frac{\Delta{v}}{\Delta{t}} = \frac{v_2 - v_1}{\Delta{t}}\;(1)

We can rearrange v_2 from equation (1) like so

\bar{a} &= \frac{v_2 - v_1}{\Delta{t}}\\ \bar{a}\Delta{t} &= v_2 - v_1\\ v_1 + \bar{a}\Delta{t} &= v_2\\ \therefore v_2 &= v_1 + \bar{a}\Delta{t}

Therefore

v_2 &= v_1 + \bar{a}\Delta{t}

Given

\bar{a} &= \frac{\Delta{v}}{\Delta{t}}\;(1)\\ \bar{v} &= \frac{\Delta{x}}{\Delta{t}}\;(2)\\ &= \frac{v_2 + v_1}{2}\;(3)

\Delta{t} from equation (1) can be rearranged as

\Delta{t} &= \frac{\Delta{v}}{\bar{a}} = \frac{v_2 - v_1}{\bar{a}}\;(4)

\Delta{x} from equation (2) can be rearranged like so

\bar{v} &= \frac{\Delta{x}}{\Delta{t}}\\ \bar{v}\Delta{t} &= \Delta{x}\\ \therefore \Delta{x} &= \bar{v}\cdot\Delta{t}

Using the following equations from above

  • \bar{v} = \frac{v_2 + v_1}{2} from equation (3)
  • \Delta{t} = \frac{v_2 - v_1}{\bar{a}} from equation (4)
\Delta{x} &= \bar{v}\cdot\Delta{t}\\ &= \frac{v_2 + v_1}{2}\cdot\frac{v_2 - v_1}{\bar{a}}\\ &= \frac{(v_2)^2 - (v_1)^2}{2\bar{a}}\;(5)

Rearranging equation (5)

2\bar{a}\Delta{x} &= (v_2)^2 - (v_1)^2

Rearrange again to obtain the more common form

(v_2)^2 &= (v_1)^2 + 2\bar{a}\Delta{x}
